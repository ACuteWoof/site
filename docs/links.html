Useful links

These are some links to pages that I find very useful. Some are pretty well known, some are pages I’ve never heard of before finding them after hours of searching for something very specific.

You can also find more useful content on my Kiwix instance: https://kiwix.lewoof.xyz.

Learning programming languages with examples

  • https://gobyexample.com/ - Learn Go with examples
  • https://www.cbyexample.com/ - Learn C and C++ with examples
  • https://www.javascriptbyexample.com/ - Learn JS with examples
  • https://rustbyexample.io/ - Learn Rust with examples
  • https://lotz84.github.io/haskellbyexample/ - Learn Haskell with examples
  • https://www.jpryan.me/dartbyexample/ - Learn Dart with examples

Consumer rights & self hosting

  • https://consumerrights.wiki/ - Louis Rossmann's consumer rights wiki
  • https://wiki.futo.org/ - FUTO Wiki (guide to a self managed life)
  • https://comfy.guide/ - Comfy self hosting guides
  • https://git.lewoof.xyz/gitbrowse - Gitbrowse, a simple web server by me for git
  • https://awesome-selfhosted.net/ - list of self-hosted alternatives to popular services
  • https://github.com/mendel5/alternative-front-ends?tab=readme-ov-file - list of alternative frontends
  • https://github.com/digitalblossom/alternative-frontends - another list of alternative frontends
  • https://sr.ht/~jamesponddotco/awesome-privacy-front-ends/ - another list of alternative frontends
  • https://libredirect.github.io/ - another list of alternative frontends and extension that autoredirects to those alternatives
  • https://farside.link/ - automatically redirects any url to its appropriate private alternative frontend
  • https://www.privacyguides.org/en/ - guides on privacy, list of alternatives, etc.
  • https://docs.invidious.io/instances/ - list of invidious instances
  • https://nitter.net - alternative frontend for Twitter, for anyone who for whatever reason wants to use Twitter
  • https://kiwix.org/ - download and host Wikipedia and other sites that collect and store knowledge
